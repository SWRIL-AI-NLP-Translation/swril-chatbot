# swril-chatbot
Chatbot React Webpack module for CDN


## Development:

### Pre-requisites:

- VSCode Live Server Extension
- Node.js / NPM

### Initialization:

Generate .env:

`cp .example.env .env`

Install NPM deps:

`npm i`

### Running:

Compile code with:

`npm run build`

Compile code on code changes (recommended) with:

`npm run dev`

Click `Go Live` to initialize Live Server
